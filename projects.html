Raktos School

Tech : Django + Jquery (Aout 2024)

Ce site web local permet aux utilisateurs d'enregistrer des cours sous forme de notes textuelles. Ronhy Raktos Word

Tech : PyQT5 (Python) (Mars 2024)

C'est un jeu de mot inspiré du jeu de pendu, le joueur tente de deviner un mot mystérieux en choisissant des lettres une par une. À chaque bonne lettre, celle-ci est révélée dans le mot, mais chaque mauvaise tentative, le joueur perd une vie. Le but est de trouver le mot avant que le nombre de vie soit nul.
